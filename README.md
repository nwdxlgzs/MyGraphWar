# MyGraphWar

一个由 Python 托管、浏览器直接游玩的数学函数炮战游戏。

## 启动

Windows PowerShell：

```powershell
.\start.ps1
```

也可以直接双击 `start.cmd`，它会绕过常见的 PowerShell 执行策略限制。启动脚本会安装缺失依赖、重新构建最新网页、启动服务器并自动打开 `http://127.0.0.1:8000`。如不希望自动打开浏览器，可运行 `.\start.ps1 -NoBrowser`。

首次启动会安装 Python/Node 依赖并构建网页。随后打开 <http://127.0.0.1:8000>，注册账号即可游玩。

局域网开放可设置 `$env:MGW_HOST="0.0.0.0"` 后启动，并在防火墙中允许端口。默认数据保存在 `mygraphwar.db`。

建议在服务器停止后备份账号、战绩和重放：

```powershell
.\backup.ps1
```

备份文件写入 `backups/`。服务监听地址、端口和数据库位置可参考 `.env.example` 配置。

## 开发

```powershell
python -m pip install -r requirements.txt
python run.py
cd web
npm install
npm run dev
```

真实浏览器端到端测试：

```powershell
cd web
npx playwright install chromium
npm run build
npm run test:e2e
```

统一验证命令：

```powershell
.\verify.ps1
.\verify.ps1 -E2E
```

导出不含本机依赖、构建产物、数据库和缓存的干净源码项目：

```powershell
python Release.py
```

输出位于 `Release-Project/`，并包含带文件哈希的 `RELEASE-MANIFEST.json`。可先使用 `python Release.py --dry-run` 查看将被导出的文件。`Release-App` 独立可执行分发包的入口已预留为 `python Release.py --app`，目前不会生成不完整的程序包。

开发网页运行在 5173 端口；生产构建由 FastAPI 直接托管。

## 原版坐标与函数模式

战场使用原版 770×450 内部绘图区，房间默认对应 `x = -25…25`、`y ≈ -14.6104…14.6104` 的平面直角坐标系。房主可输入 `1～100` 之间的任意横轴半范围（支持小数），也可选择每局在 `±5～±50` 间自动随机；纵轴、障碍物、士兵/爆炸半径和模拟采样精度会同步缩放，因此这会真正改变函数坐标单位，而不只是缩放 Canvas。地图由圆形、椭圆、矩形和菱形等随机障碍组成，所有形状都参与服务器权威碰撞；函数撞击后会炸出局部圆形缺口，而不会整块消失。每位玩家可独立配置 1–4 名士兵，默认两名；士兵随机分布在所属队伍的整个半场。函数直接穿过士兵即将其击杀，但会继续前进并可连续命中多人。

创建房间时，“经典规则”遵循原版的 A/B 团队战和最多 10 名玩家；“扩展规则”提供自由混战和最多 20 名参与者。点击开始后由服务器执行 5 秒同步倒计时，期间锁定席位与设置。

- 普通函数：输入 `y=f(x)`，服务器仅沿 y 轴平移曲线，使它经过当前士兵；士兵不在原点。
- 一阶微分方程：输入 `y'=f(x,y)`，士兵坐标作为初始条件。
- 二阶微分方程：输入 `y''=f(x,y,y')`，士兵坐标与发射角作为初始条件。

普通函数使用自适应步长绘制；微分方程使用四阶 Runge–Kutta 数值积分。右侧队伍会镜像坐标，因此双方以相同函数获得对称轨迹。每回合限时 60 秒，射击后有 3 秒结算阶段。

扩展规则额外支持：

- 极坐标 `r=f(theta)`：以当前士兵为局部极点，将半径平移到从士兵圆周出发，右半场玩家自动镜像。
- 参数方程 `x=f(t), y=g(t)`：两式在 `t=0` 时整体平移到当前士兵，以服务器固定步长绘制并使用同一套障碍和士兵碰撞。

两种扩展模式同样支持 AI、观战、重连、战绩和完整重放，不会出现在经典规则房间中。

正在进行的房间可由大厅直接进入观战；观战者只能查看和聊天。完整比赛、参与者与版本化重放会永久保存；大厅可以查看个人战绩、最近比赛，并在浏览器中逐发或自动播放历史重放。

大厅也可以导入先前下载或备份的 MyGraphWar v1 JSON 重放文件，无需原房间或原服务器数据库即可播放。

## 函数语法

变量为 `x`、`y`、`y'`（具体取决于模式），支持 `+ - * / % ^`、括号、`sin cos tan asin acos atan sqrt abs exp log ln floor ceil`、`pi` 和 `e`。兼容原版的 `sen`、`tg`、小数逗号以及 `2x`、`3(x+1)` 隐式乘法；`log` 为十进制对数，`ln` 为自然对数。二阶模式角度属于每名士兵，可用上下方向键调整并在后续回合保留。
