# MyGraphWar 实施计划

MyGraphWar 是一个浏览器端数学炮战游戏。FastAPI 服务器负责账号、房间、WebSocket、权威轨迹与结算；React + Canvas 负责界面和动画；SQLite 保存账号和战绩。

## 核心能力

- 注册、登录、持久会话和战绩。
- 2–20 个真人/AI，团队战或自由混战。
- 默认使用原版 `x=-25…25`、`y≈-14.6…14.6` 平面直角坐标战场，并支持房间级任意坐标半范围及每局随机范围；圆形、椭圆、矩形、菱形障碍、单位、局部爆炸缺口和采样精度随坐标比例缩放。
- 普通函数纵向平移、一阶微分方程、二阶微分方程和发射角。
- 安全数学表达式、确定性数值积分、直接命中连续击杀、障碍碰撞与 60 秒回合限制。
- WebSocket 房间状态、聊天、准备、回合和重连快照。
- 简单、中等、困难、自适应四档 AI。
- Python 单进程托管生产网页，保留版本化 REST/WebSocket 接口。

## 架构

- `server/`：配置、数据库、认证、房间、模拟、AI 和 FastAPI 应用。
- `web/`：React、TypeScript、Vite 与 Canvas 游戏客户端。
- `tests/`：表达式、模拟、认证和 API 回归测试。
- `start.ps1`：安装/构建后启动本机服务器。

## 验收

浏览器注册后可创建房间，配置混战/团队、添加 AI、开始对局，并在真实坐标平面上用三种经典函数模式绕过障碍物攻击；服务器重启后账号、战绩与重放保留；非法表达式和越权回合被拒绝。扩展规则另含极坐标、参数方程、20 人自由混战，且不改变经典规则。
